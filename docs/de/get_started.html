    1. QGIS 3 Installieren, bis zum offiziellen Release von QGIS 3 kann QGIS 2.99 von OSGeo4W oder einer für die Plattform geeigneten Quelle installiert werden.

    2. Die QgsProjectGenerator Erweiterung über das Erweiterungsmenü installieren.

    3. Das Datenbank –> Projektgenerator –> Generieren Menü anwählen

    4. Die Datenbank und - falls zutreffend - Interlis konfigurieren
